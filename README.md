# Приложение "Моя Смена"

Мобильное Android-приложение для управления рабочими сменами и отслеживания производительности сотрудников в текстильной промышленности.

## Описание

Приложение "Моя Смена" предназначено для автоматизации процессов учета рабочего времени, отслеживания производительности и управления задачами на производстве. Оно предоставляет индивидуальные дашборды для разных ролей сотрудников: рабочих, мастеров, кладовщиков и менеджеров.

## Основные функции

### Для рабочих:
- Просмотр текущих задач и заданий
- Отслеживание выполненных операций
- Контроль качества продукции
- Статистика личной производительности

### Для мастеров:
- Мониторинг работы бригады
- Распределение задач между рабочими
- Контроль качества выполненных работ
- Анализ производительности подчиненных

### Для кладовщиков:
- Управление материалами и комплектующими
- Отслеживание расхода ресурсов
- Учет поступлений и списаний
- Формирование отчетов о наличии материалов

### Для менеджеров:
- Обзор ключевых показателей эффективности (KPI)
- Анализ производительности всех бригад
- Отслеживание выполнения заказов
- Мониторинг прогресса проектов

## Технические характеристики

- **Платформа:** Android 7.0 (API level 24) и выше
- **Язык программирования:** Java/Kotlin
- **База данных:** SQLite
- **Архитектура:** Многоуровневая архитектура с разделением по ролям

## Установка и запуск

1. Клонируйте репозиторий:
   ```
   git clone <repository-url>
   ```

2. Откройте проект в Android Studio

3. Соберите проект:
   ```
   ./gradlew build
   ```

4. Запустите приложение на эмуляторе или реальном устройстве:
   ```
   ./gradlew installDebug
   ```

## Структура проекта

```
app/
├── src/main/java/com/example/clothes/
│   ├── MainActivity.java          # Экран входа в систему
│   ├── DashboardActivity.java      # Дашборд для рабочих, мастеров и менеджеров
│   ├── DashboardStorekeeper.java  # Дашборд для кладовщиков
│   ├── DatabaseHelper.java        # Работа с локальной базой данных SQLite

│   └── StorekeeperStats.java      # Статистика для кладовщиков
├── src/main/res/                  # Ресурсы приложения (layouts, strings, etc.)
└── build.gradle.kts              # Конфигурационный файл сборки
```
